# 📊 Proyecto de Estadística I — Análisis de Población Mundial (2026)

## 🌍 Descripción

Este proyecto es un dashboard interactivo desarrollado con **Streamlit** que analiza datos de población, área territorial, densidad poblacional y porcentaje mundial de países y territorios proyectados para el año 2026.

El dataset incluye únicamente:
- País
- Población (2026)
- Área (km²)

A partir de estos datos se calculan las siguientes variables:
- Densidad poblacional
- Porcentaje de población mundial
- Ranking por población
